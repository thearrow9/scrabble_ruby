Scrabble CLI
-------------

### Инсталиране ###

Стартирайте `ruby install.rb` и след това `ruby scrabble.rb`

### Поддържани езици за играта ###

* английски
* български

### Избор на език за игра ###

Променете ред 1 на файла `lib/constants.rb`
